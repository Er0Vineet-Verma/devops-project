Project Overview

Welcome to the Spotify-style DevOps Dashboard project. This single-page application demonstrates a modern CI/CD pipeline using Docker, Minikube, and VS Code Dev Containers.
Scroll through to see how each technology fits into the workflow.

Key Features

  • Containerised Development: Leverage Docker Compose & Dev Containers for reproducible environments.
  • Kubernetes-ready: Local clusters powered by Minikube mirror production setups.
  • GitHub Actions CI: Automated testing & image publishing on every pull request.
  • Helm Charts: Declarative application packaging for effortless releases.

Technology Stack

Frontend

HTML 5, CSS 3 (Spotify-inspired) & Vanilla JS.

Containerisation

Dockerfile & Docker Compose for local builds.

Orchestration

Minikube + Helm for local Kubernetes clusters.

CI/CD

GitHub Actions workflows for automated testing & deployment.

Local Deployment Guide

  1. Clone the repo: git clone https://github.com/your-org/spotify-devops-dashboard.git
  2. Build the image: docker compose build
  3. Run locally: docker compose up
  4. Spin up Minikube: minikube start
  5. Install via Helm: helm install dashboard ./charts/dashboard

Once deployed, access the dashboard at http://localhost:3000 and enjoy monitoring in true Spotify style!
